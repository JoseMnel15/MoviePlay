Para el desarrollo de este proyecto se va a utilizar un editor de codigo llamado Visual Studio Code. 
Las tecnologias a utilizar son: HTML5, CSS y Javascript como Fronted. En backend utilizaremos un gestor de bases de datos MySql y Java. 
Para crear las interfaces (vistas del proyecto) se utilizara Adobe XD (adobe experience design). 
El navegador con que estare trabajando va a ser Google Chrome.
